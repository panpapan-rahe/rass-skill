# Frontend Reset Protocol

This note records the reset performed in the session where the user asked to start frontend learning from zero.

## What was removed
- Obsolete frontend reference files were deleted from the umbrella skill.
- The frontend learning cron job was removed.
- The frontend subtree under the previous split skills was cleared.

## What remained
- The root umbrella skill stayed intact as the class-level home for future frontend/backend learning.
- Backend learning remained untouched.

## Rebuild rule
When restarting frontend learning:
1. Start with a clean subtree.
2. Reintroduce only the new frontend curriculum.
3. Avoid reusing stale frontend notes from the prior learning path.
